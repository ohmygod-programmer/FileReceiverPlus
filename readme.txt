Запуск осуществляется с помощью gradlew.bat run
Либо собрать с помощью gradlew.bat jar и запустить с помощью java -jar <имя файла>.jar
При запуске есть несколько обязательных параметров.

Первый обязательный параметр определяет в каком режиме запустится приложение:
"client" запустит приложение как клиент, "server" - соответственно как сервер.
Далее, для клиента еще 3 обязательных параметра:
1) ip адрес сервера для подключения
2) порт сервера
3) Имя/путь файла для отправки на сервер

Для сервера же 1 обязательный параметр и 1 необязательный:
1) порт для прослушивания
2) ip адрес адаптера, через который процесс будет взаимодействовать с сетью.
Второй параметр необязателен. Если не указать, адаптер выбирается автоматически.

Все параметры вводятся последовательно через пробел при запуске программы
